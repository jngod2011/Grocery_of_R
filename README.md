# Grocery_of_R
A Grocery of R

A Grocery of R, is a book on R applications for beginners, with brief introduction and verified codes on:

1.Data manipulation

2.Beginner's Statistics

3.Machine Learning

4.MCMC

5.Time Series, Chaos

6.Economy

7.Clinical, Epidemic, Bio

8.Plotting

Note: The book is written in simplified Chinese.
